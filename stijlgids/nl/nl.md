## Valuta

### Standaard

Gebruik `EUR` met spatie: `9,99 EUR` `2,5 EUR/stuk`

### Symbool

Symbool **VÓÓR** getal met spatie:

```
✅ € 83  | £ 1.803,75  | € 8,17/m²
❌ 83 €  | 83€         | 8,17€/m²
```

## Leestekens

### Spaties

**GEEN** spatie vóór: `:` `;` `?` `!`

```
✅ Vraag: wat is dit?
❌ Vraag : wat is dit ?
```

### Aanhalingstekens

- Citaten: dubbel `"..."`
- Anders: enkel `'...'`
- Geen spatie voor/na: `"juist"` `(juist)` `[juist]`

### Opsommingen

Structuur:
```
Inleiding:
- eerste punt;
- tweede punt;
- laatste punt.
```

Regels:
- Begin elk punt: kleine letter
- Einde elk punt: puntkomma `;`
- Laatste punt: punt `.`

```
✅ Producten:
   - cellulose-isolatie;
   - houtvezelplaten;
   - testapparatuur.

❌ Producten:
   - Cellulose-isolatie.
   - Houtvezelplaten.
   - Testapparatuur.
```

### Voetnoten

Nummer **na** punt (optioneel vetgedrukt)
