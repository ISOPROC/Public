## Getallen / Nombres

### Decimalen en duizendtallen

- Decimaalteken: komma `,`
- Duizendtallen: punt `.`
- Negatief: geen spatie na `-`

```
вң… 3,14  | 12.735,52  | -1.500
вқҢ 3.14  | 12,735.52  | - 1.500
```

### Formules

Spatie rond operatoren: `+` `-` `Г—` `/` `=` `<` `>` `вүӨ` `вүҘ` `вү ` `Вұ`

```
вң… 3 + 2 = 5  | 7 Г— -0,5 = -3,5
вқҢ 3+2=5      | 7Г—-0,5=-3,5
```

### Benadering

Spatie na `+/-`

```
вң… +/- 3,14
вқҢ +/-3,14
```

## Eenheden / UnitГ©s

### Basisregel

**GEEN spatie** tussen getal en eenheid (symbool/afkorting)

```
вң… 3m  | 19В°C  | 3m/s  | 60mВІ/pallet
вқҢ 3 m | 19 В°C | 3 m/s | 60 mВІ/pallet
```

### Uitzonderingen

**WEL spatie** bij:
- Voluit geschreven eenheden: `3 jaar` `5 meter` / `3 ans` `5 mГЁtres`
- Valuta: `9,99 EUR` `2,5 EUR/stuk`

### Samengestelde eenheden

- Deling: `/` zonder spaties: `3m/s` `12 rollen/karton`
- Vermenigvuldiging: `.` zonder spaties: `1N.m`
- Complexe: gebruik steeds haakjes zonder spaties bij breuken met gecombineerde eenheden in teller of noemer: `W/(m.K)` `(mВІ.K)/W`

### Machten

Gebruik superscript: `mВІ` `mВі` (niet `m2` `m3`)

### Tabellen

Eenheid tussen rechte haakjes: `[m]` `[EUR/mВІ]` `[(mВІ.K)/W]`

## Hoofdletters / Majuscules

### Basisregel

Bij twijfel: **kleine letter**

### Merknamen

Originele schrijfwijze behouden, ook aan begin van een zin:

```
вң… pro clima blijft altijd klein / pro clima reste toujours en minuscules
вқҢ Pro clima | Pro Clima | PRO CLIMA
```

```
вң… De pro clima SOLITEX MENTO 3000 is waterdicht.
вң… Le pro clima SOLITEX MENTO 3000 est Г©tanche.
```

### Na dubbele punt

**Geen** hoofdletter na `:`

```
вң… Producten: cellulose, houtvezel
вқҢ Producten: Cellulose, Houtvezel
```
