## Getallen / Nombres

### Decimalen en duizendtallen

- Decimaalteken: komma `,`
- Duizendtallen: punt `.`
- Negatief: geen spatie na `-`

```
✅ 3,14  | 12.735,52  | -1.500
❌ 3.14  | 12,735.52  | - 1.500
```

### Formules

Spatie rond operatoren: `+` `-` `×` `/` `=` `<` `>` `≤` `≥` `≠` `±`

```
✅ 3 + 2 = 5  | 7 × -0,5 = -3,5
❌ 3+2=5      | 7×-0,5=-3,5
```

### Benadering

Spatie na `+/-`

```
✅ +/- 3,14
❌ +/-3,14
```

## Eenheden / Unités

### Basisregel

**GEEN spatie** tussen getal en eenheid (symbool/afkorting)

```
✅ 3m  | 19°C  | 3m/s  | 60m²/pallet
❌ 3 m | 19 °C | 3 m/s | 60 m²/pallet
```

### Uitzonderingen

**WEL spatie** bij:
- Voluit geschreven eenheden: `3 jaar` `5 meter` / `3 ans` `5 mètres`
- Valuta: `9,99 EUR` `2,5 EUR/stuk`

### Samengestelde eenheden

- Deling: `/` zonder spaties: `3m/s` `12 rollen/karton`
- Vermenigvuldiging: `.` zonder spaties: `1N.m`
- Complexe: gebruik steeds haakjes zonder spaties bij breuken met gecombineerde eenheden in teller of noemer: `W/(m.K)` `(m².K)/W`

### Machten

Gebruik superscript: `m²` `m³` (niet `m2` `m3`)

### Tabellen

Eenheid tussen rechte haakjes: `[m]` `[EUR/m²]` `[(m².K)/W]`

## Hoofdletters / Majuscules

### Basisregel

Bij twijfel: **kleine letter**

### Merknamen

Originele schrijfwijze behouden, ook aan begin van een zin:

```
✅ pro clima blijft altijd klein / pro clima reste toujours en minuscules
❌ Pro clima | Pro Clima | PRO CLIMA
```

```
✅ De pro clima SOLITEX MENTO 3000 is waterdicht.
✅ Le pro clima SOLITEX MENTO 3000 est étanche.
```

### Na dubbele punt

**Geen** hoofdletter na `:`

```
✅ Producten: cellulose, houtvezel
❌ Producten: Cellulose, Houtvezel
```

## Koppelteken / Trait d'union

### Merknaam + product

Koppelteken als merknaam vóór product:

```
✅ iQ3-cellulose
✅ Celit-houtvezelplaat / Celit-panneau de fibres de bois
✅ Celit 4D-onderdakplaat / Celit 4D-panneau sous-toiture

❌ iQ3 cellulose
❌ Celit houtvezelplaat
```

Geen koppelteken als merknaam ná product:

```
✅ cellulose-isolatie iQ3 / ouate de cellulose iQ3
✅ isolerende onderdakplaat Celit 4D / panneau isolant sous-toiture Celit 4D

❌ cellulose-isolatie-iQ3
❌ isolerende-onderdakplaat-Celit-4D
```
