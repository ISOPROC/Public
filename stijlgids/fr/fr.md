## Devises

### Standard

Utiliser `EUR` avec espace : `9,99 EUR` `2,5 EUR/pièce`

### Symbole

Symbole **APRÈS** le nombre avec espace :

```
✅ 0,95 €  | 1.803,75 £  | 8,17 €/m²
❌ € 0,95  | £1.803,75   | €8,17/m²
```

## Ponctuation

### Espaces

**AVEC espace** avant : `:` `;` `?` `!`

```
✅ Question : qu'est-ce que c'est ?
❌ Question: qu'est-ce que c'est?
```

### Guillemets

- Citations : guillemets français `« ... »` (avec espaces insécables)
- Pas d'espace avant/après : `« correct »` `(correct)` `[correct]`

### Énumérations

Structure :
```
Introduction :
- premier point ;
- deuxième point ;
- dernier point.
```

Règles :
- Début de chaque point : minuscule
- Fin de chaque point : point-virgule `;`
- Dernier point : point `.`

```
✅ Produits :
   - isolation ouate de cellulose ;
   - panneaux de fibres de bois ;
   - équipement de test.

❌ Produits :
   - Isolation ouate en cellulose.
   - Panneaux de fibres de bois.
   - Équipement de test.
```

### Notes de bas de page

Numéro **après** le point (gras optionnel)

❌ valeur µd | μd | µd
